# Flow
Application for listening audio files on vk.com (Vkontakte)

Created by using Qt5 and VK API.

It has user-friendly interface and very beautiful design.

### Required access
-  Audio files
-  Anytime (for playing audio while offline)

### Features
-  Searching
-  Downloading
-  Shuffling
-  Repeat modes
